Installation

Uploading via FTP

  • Download the zipped theme pack to your computer from themeforest and extract the ZIP file to a folder on your computer.
  • In the extracted folder you will find 'the-rex.zip'. Extract this file .
  • Using an FTP client to access your host web server
  • Upload the Theme files in the extracted folder called 'the-rex' to wp-content/themes directory provided by WordPress.
  • Visit Admin panel -> Appearance -> Themes, look for the REX theme thumbnail, then click the Activate button to activate the theme.

Installing From the WordPress Dashboard

  • Download the zipped theme pack to your local computer from themeforest and extract the ZIP file contents to a folder on your local computer.
  • Log into your WordPress admin panel and click on the “Appearance” tab.
  • Now click on the "Add New" button.
  • Click on the "Upload" tab, click "Choose File" and select the-rex.zip in the extracted folder then click "Open".
  • Click the "Install Now" button.
  • Activate the theme from this page by clicking "Activate", or "Return to Themes Page" and activate from there

Installing Demo Content

Manually Upload XML File

This Method will import posts, pages, menus only not the demo site theme settings.

  • Go to Admin panel -> Tools -> Import -> WordPress
  • Install the plugin.
  • Use it to upload the demo content XML file in the theme package.

1. Go to Dashboard > Tools > Import and Click on Wordpress and Install it.

2. Choose a content_data.xml file from your download package and then click on Upload file and Import

3. Assign Authors and start import, click on Submit

4. This process may take a while on some hosts, so please be patient. (some hosts between 5-6 minutes)

Theme Navigations

Setup A New Menu

  1. Navigate to Apperance -> Menus page
  2. Click the "create a new menu" link to make a new menu. Enter the name then hit Create Menu
  3. To add a menu item, you can easily select one of your already made pages on the left hand side and click Add to Menu. You can also add menu items by entering a custom name and custom link into the Links box. To create a megamenu, see Categories Mega Menus.
  4. You can manage your menus using the drag and drop functionality. To create a dropdown menu, simply drag a menu item below and slightly to the right of another menu item and it will lock into place and create a dropdown section.
  5. After setting up your menu, select menu location in the Menu Settings (Top Menu for navigation in top bar and Main Menu for main navigation below header)
  6. Once its all done, make sure you click the Save button.
  7. You can change menus locations at any time from Manage Locations tab .

Categories Mega Menus

The theme features a mega menu for category menu items to display latest posts in the corresponding category

  • Add the Category you want to the menu.
  • In the setting panel of category menu item there will be a checkbox called BK Megamenu.
  • Check it if you want to make that category menu item a megamenu or uncheck to make it a normal menu item.
  • Don't forget to set the menu location to Main Menu by clicking checkbox Main Menu in Theme locations section of Menu Settings.
  • Click Save Menu to finish.
Both megamenu and normal menu items can have child menu.The maximum level of child menu to display in Main Menu is 2 level down, and 1 level for Top Menu.

Post format

The theme support 5 WordPress post formats: Standard, Gallery, Video, Image, Audio.

Add or edit format of a post

  • In post edit page, choose post format in the Format box on the right.
  • Enter information for the corresponding format options in BK Post format options box:
    • Post format video and audio: enter the url of the video or audio item. The theme support link from Youtube, Vimeo and Souncloud.
    • Post format image: click Select files button to select images or drag images in to image upload area to upload image to the post.
    • Post format gallery: click Select or Upload images button to add a gallerry to the post.

Adding a Review Post

  1. From the BK Review System box in the post edit page .
  2. Tick the checkbox Enable Review On This Post.
  3. Enter the criteria name and use the slider to adjust rating score. The Final average score will be calculated automatically.
  4. You can have up to 6 criteria in a review box.
  5. Type the Summary textarea your summary for the review item. You can create lists of pros and cons in here.
  6. Check to enable user review
  7. Setup position for review box in single page
  8. Click Publish or Update button to save.

Blog and Archive layout

The theme has 6 blog layouts that can be set seperately for each page and category.

  1. Classic Blog: Blog layout with small thumbnail. This layout has sidebar.

  2. Large Blog: Posts are displayed with large thumbnail. This layout has sidebar.

  3. Masonry with sidebar: Posts are displayed in a two columns Masonry layout. This layout has sidebar.

  4. Masonry without sidebar: Posts are displayed in a two columns Masonry layout. This layout has no sidebar.

  5. Square Grid with sidebar: Posts are displayed in a two columns Masonry layout. This layout has sidebar.

  6. Square Grid without sidebar: Posts are displayed in a two columns Masonry layout. This layout has no sidebar.

Setting page layout

  1. In Admin panel, go to Theme Options -> Page Layout tab
  2. For each page listed, choose a desire layout and sidebar.
  3. Click Save to save option.
The layout chosen for Category is applied to all category and will be overidden by layout option set in Category edit page. To edit a specific category layout, see Category layout

Shortcode

The theme supports the following shortcodes that can be used in shortcode module (You must install and activate the bk_shortcode plugin comes with the theme before using this function)

You can also use these shortcodes in post editor (without [bkend] at the end of each shortcode)

  1. Youtube Video Shortcode

    [youtube_embed src="PUT YOUR YOUTUBE URL HERE"/]
[bkend]

    Example:

    [youtube_embed src="https://www.youtube.com/watch?v=e-ORhEE9VVg"/]
[bkend]

  2. Vimeo Video Shortcode

    [vimeo_embed src="PUT YOUR VIMEO URL HERE"/]
[bkend]

    Example:

    [vimeo_embed src="https://vimeo.com/108210854"/]
[bkend]

  3. One Half Shortcode

    [one_halfs]
[one_half]PUT YOUR CONTENT FOR THE FIRST COLUMN[/one_half] 
[one_half]PUT YOUR CONTENT FOR THE LAST COLUMN[/one_half]
[/one_halfs]
[bkend]

    Example:

    [one_halfs]
[one_half]Ultrices habitant sociosqu leo vestibulum. [/one_half] 
[one_half]Id id condimentum sed ut mattis aenean dis sollicitudin sed volutpat[/one_half]
[/one_halfs]
[bkend]

  4. 1/3 1/3 1/3 Column Shortcode

    [one_thirds]
[one_third]PUT YOUR CONTENT FOR THE FIRST COLUMN[/one_third] 
[one_third]PUT YOUR CONTENT FOR THE SECOND COLUMN[/one_third] 
[one_third]PUT YOUR CONTENT FOR THE LAST COLUMN[/one_third] 
[/one_thirds]
[bkend]

    Example:

    [one_thirds]
[one_third]Ultrices habitant sociosqu leo vestibulum.[/one_third] 
[one_third]Ultrices habitant sociosqu leo vestibulum. [/one_third] 
[one_third]Ultrices habitant sociosqu leo vestibulum. [/one_third] 
[/one_thirds]
[bkend]

  5. 2/3 1/3 Column Shortcode

    [twothird_onethird]
[two_third]PUT YOUR CONTENT FOR THE TWO THIRD COLUMN[/two_third]
[one_third]PUT YOUR CONTENT FOR THE ONE THIRD COLUMN[/one_third]
[/twothird_onethird]
[bkend]

    Example:

    [twothird_onethird]
[two_third]Ultrices habitant sociosqu leo vestibulum. [/two_third]
[one_third]Ultrices habitant sociosqu leo vestibulum. [/one_third]
[/twothird_onethird]
[bkend]

  6. Info Box Shortcode

    [bkinfobox backgroundcolor="PUT YOUR COLOR CODE" title="PUT YOUR TITLE"]PUT YOUR CONTENT[/bkinfobox]
[bkend]

    Example:

    [bkinfobox title="Info box" backgroundcolor="#F8F8F8" ]Ultrices habitant sociosqu leo vestibulum. Euismod, fermentum per aptent Nibh dictumst lorem quis viverra[/bkinfobox]
[bkend]

  7. Accordion Shortcode

    [accordions]
[accordion title="PUT YOUR TITLE"]PUT YOUR CONTENT[/accordion]
...
[/accordions]

    Note: You can put multi accordion shortcodes. See the example below for 3 consecutive accordion shortcodes

    [accordions]
[accordion title="Accordion Title 0"]Accordion Content 0[/accordion]
[accordion title="Accordion Title 1"]Accordion Content 1[/accordion]
[accordion title="Accordion Title 2"]Accordion Content 2[/accordion]
[/accordions]

  8. Tabs Shortcode

    [tabs]
[tab title="PUT YOUR TAB TITLE"]PUT YOUR TAB CONTENT[/tab]
...
[/tabs]
[/bksctabs]
[bkend]

    Example:

    [tabs]
[tab title="Tab Title 0"]Tab Content 0[/tab]
[tab title="Tab Title 1"]Tab Content 1[/tab]
[tab title="Tab Title 2"]Tab Content 2[/tab]
[/tabs]
[bkend]

  9. Parallax Image Shortcode

    [bkparallax image_url = "PUT YOUR IMAGE URL" url = "PUT YOUR URL" title = "PUT YOUR TITLE" text_desc= "PUT YOUR DESCRIPTION"][/bkparallax]
[bkend]

    Example:

    [bkparallax image_url = "http://localhost/wordpress/wp-content/uploads/2015/01/2390379489_4ab06f3536_b-.jpg" url = "sense.bk-ninja.com" title = "Parallax Image" text_desc= "Mauris viverra nunc congue augue euismod suscipit. Class aptent taciti sociosqu ad litora torquent per conubia nostra, per inceptos himenaeos"][/bkparallax]
[bkend]

  10. Team Shortcode

    [bkteam number="PUT YOUR NUMBER OF AUTHOR TO BE SHOWED" item_in_row = "PUT THE NUMBER OF ITEMS IN A ROW (Recommend: put 3 for the content has sidebar section and 4 for the full width section)"][/bkteam]
[bkend]

    Example:

    [bkteam number="6" item_in_row = "4"][/bkteam]
[bkend]

  11. Author Box Shortcode

    [authorbox authorid = "PUT THE AUTHOR ID"][/authorbox]
[bkend]

    Example:

    [authorbox authorid = "1"][/authorbox]

  12. Button Shortcode

    [buttons]
[button href="PUT THE LINK HERE" name="PUT THE BUTTON NAME" backgroundcolor="PUT THE BACKGROUND COLOR CODE" textcolor="PUT THE TEXT COLOR CODE"]
...
[/buttons]
[bkend]

    Example:

    [buttons]
[button href="#" name="Button 1" backgroundcolor="#FE6969" textcolor="#fff"]
[button href="#" name="Button 2" backgroundcolor="#FACE00" textcolor="#fff"]
[/buttons]

  13. Google Adsense Shortcode

    Example 1: Float Left

    [bkga float="left"]Google Adsense Code[/bkga]

    Example 2: Float Right

    [bkga float="right"]Google Adsense Code[/bkga]

Updating Wordpress Theme

This theme may be updated periodically in order to add new features and fix bugs. When a new update be available an update message will appears on your WordPress dashboard . As a customer, you can download the most current version from your ThemeForest account.

Updating Automatically

Envato have developed the Envato Toolkit Plugin for WordPress. You can use this to receive notification of updates to themes purchased from ThemeForest and to automatically update (upon clicking) directly from within your WordPress admin area.

  • Download the plugin zip file to your computer.
  • In your WordPress admin area, go to Plugins -> Add New then click Upload.
  • Click Choose File, select the plugin zip file you downloaded and click Install Now.
  • After installation, click Activate.
  • Click the new Envato Toolkit link in the menu and follow the instructions to configure the plugin.

Updating Manually

  • Download the most current version from ThemeForest in the "Downloads" area of your account.
  • Unzip the package and locate the-rex.zip in the theme folder.
  • Go to Appearance -> Themes and activate another theme such as the default WordPress theme.
  • Delete the the-rex theme which is now inactive.
  • Go to Install Themes -> Upload then install and activate the-rex.zip from the new package.

Language Translations

The REX is able to be translated into any language. The process of translating the theme is a user responsibility. The following instructions are given as guidance.

  1. Open wp-config.php and replace this: define ('WPLANG', ''); with this (subsitute the language string (bg_BG) with your own!): define ('WPLANG', 'bg_BG');
  2. Download and install POEDIT
  3. Connect to your site -> open your theme/languages directory
  4. Look for the default.po file in theme package and open it with POEDIT.
  5. Translate file and save it as bg_BG (the file name must match with the string you inserted into wp-config.php)
  6. Two files will be generated after save. bg_BG.po and bg_BG.mo
  7. Upload the .mo and .po files into wp-content/themes/your-theme/languages folder
Your WordPress installation should be in the language you intend on using the site.
